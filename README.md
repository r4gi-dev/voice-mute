# voice-mute

FiveM用のシンプルなマイクミュートスクリプト。
近距離ボイスのみをミュートし、無線（ラジオ）には影響しません。

## 🔧 機能

* トグル式ミュート
* 押している間だけミュート（Push-to-Mute）
* 右下にミュート状態を表示
* ミュートON/OFF時に効果音
* リソース停止時の自動リセット

## 🎮 操作方法

| キー | 内容                 |
| -- | ------------------ |
| M  | ミュート ON / OFF 切り替え |
| N  | 押している間だけミュート       |

※キーはFiveMのキーバインド設定から変更可能

## ⚙️ 仕様

* 近距離ボイス（proximity）のみミュート
* 無線（radio / channel）は影響なし
* pma-voiceの `overrideProximityCheck` を使用

## 📦 必要要件

* pma-voice

## 🚀 インストール

1. `resources` フォルダに配置
2. `server.cfg` に追加

```
ensure voice-mute
```

## ⚠️ 注意

* 他のボイススクリプトと競合する可能性があります

## 📝 ライセンス

自由に使用・改変OK
